## 🎮 Сюжет

> *«Завтра будет печаль…»*

> Исследуйте мир, общайтесь с персонажами и находите свой путь в этой атмосферной истории.

## 📂 Структура проекта

- `game/` — основные файлы игры:
  - `script.rpy` — основной сценарий
  - `screens.rpy` — экраны интерфейса
  - `gui.rpy` — настройки графического интерфейса
  - `options.rpy` — параметры игры
  - `yinsh.rpy` — дополнительный сценарий (возможно, отдельная ветка)
  - `images/` — спрайты, фоны, CG
  - `audio/` — музыка и звуки
  - `gui/` — элементы GUI (шрифты, кнопки)
  - `tl/` — переводы (если есть)

## 🚀 Запуск

1. Скачайте и установите [Ren'Py](https://www.renpy.org/) (версия 7.4 или выше).
2. Склонируйте репозиторий или скачайте архив.
3. В Ren'Py выберите **«Launch Project»** и укажите путь к папке с игрой (где лежит `game/`).
4. Нажмите **«Launch»** — игра запустится.

Или, если вы используете Ren'Py SDK, просто откройте проект через интерфейс.

## 🛠 Разработка

- Игра создана на Ren'Py.
- Скрипты написаны на языке Ren'Py (Python-подобный синтаксис).
- Для редактирования сценария используйте любой текстовый редактор (VS Code с плагином Ren'Py, Atom и т.п.).
